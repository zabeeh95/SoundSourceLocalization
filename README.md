**📡 Acoustic Sound Source Localization**

A practical implementation of a sound source localization system using distributed acoustic sensors. This project was developed to understand triangulation algorithms, signal-based positioning, and real-world hardware–software integration.

**🚀 Project Overview**

This project focuses on detecting and estimating the position of a sound source using:

XBee-based sound sensors

Arduino for data acquisition

MATLAB for algorithm implementation and visualization

The system collects sound intensity data from multiple sensor nodes and applies triangulation algorithms to compute the estimated coordinates of the sound source.

**🛠 System Architecture**

Hardware Layer

XBee sound sensor modules

Arduino microcontroller for data collection and communication

Algorithm Layer

Triangulation-based localization algorithm

Distance estimation from sensor readings

Coordinate computation using geometric methods

Visualization Layer

MATLAB-based plotting of:

Sensor positions

Estimated sound source location

Geometric intersection points
